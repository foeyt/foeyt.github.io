注释

注释正如其名，对一行代码进行解释，在解释器运行代码过程中，会自动省略注释。

注释分为单行注释与多行注释。

在写代码过程中，推荐添加注释，不仅让人了解你的 API 使用方法，也整理了自己的思路。

变量

变量相当于数学中的未知数，可以为任何值。

新建变量后，解释器会在系统内存中，划分一段内存，并将值储存在内存中。当调用变量时，解释器就会在对应变量的地址内返回对应值（可能有点不准确，我也是编程新手）

在 Python 中，变量不需要指定类型，只需要赋值就可以了。但也带来的不良后果，例如在 Java 这种编译型语言中，可以提前好指定类型，不需要进行赋值，而在Python 中，新建变量也必须赋值。
