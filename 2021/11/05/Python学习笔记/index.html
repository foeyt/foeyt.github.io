注释

注释正如其名，对一行代码进行解释，在解释器运行代码过程中，会自动省略注释。

注释分为单行注释与多行注释。

在写代码过程中，推荐添加注释，不仅让人了解你的 API 使用方法，也整理了自己的思路。

变量

变量相当于数学中的未知数，可以为任何值。

新建变量后，解释器会在系统内存中，划分一段内存，并将值储存在内存中。当调用变量时，解释器就会在对应变量的地址内返回对应值（可能有点不准确，我也是编程新手）

在 Python 中，变量不需要指定类型，只需要赋值就可以了。但也带来的不良后果，例如在 Java 这种编译型语言中，可以提前好指定类型，不需要进行赋值，而在Python 中，新建变量也必须赋值。

变量起名也有规范：只能使用下划线、数字、字母，且数字不能开头，变量名不能为字面值（关键字）

在编写代码过程中，对于变量的起名遵循下划线命名格式，如：hello_world, is_at_all

基础数据类型

Python 有六个基本数据类型，分别为：数字（Number）， 字符串（String）， 列表（List）， 元组（Tuple），集合（Set），字典（Dictionary）。

其中不可变数据为：数字（Number），字符串（String），元组（Tuple）。

可变数据为：列表（List），集合（Set），字典（Dictionary）。

数字（Number）

数字（Number）中有以下几种类型：整数（int），浮点数（float），布尔数（bool），复数（complex）

在 Python 中，没有 long, short, byte 等类型，但是可以进行类型转换
