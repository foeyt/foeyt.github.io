证明 Binet-Cauchy 公式

定义矩阵 A 与 B，则有

  • 若 A, B ∈ 𝔽𝕟 × 𝕟，则 det AB = det A ⋅ det B；
  • 若 A ∈ 𝔽𝕞 × 𝕟, B ∈ 𝔽𝕟 × 𝕞, m > n，则 det AB = 0；
  • 若 A ∈ 𝔽𝕞 × 𝕟, B ∈ 𝔽𝕟 × 𝕞, m ≤ n，则 det AB = ∑Sdet AS ⋅ det BS，其中 S 是 {1, 2, ⋯, n} 的一个含有 m 个元素的子集遍历，AS 为 A 的一个 m × m 子矩阵。

以上称为 Binet-Cauchy 公式。

实数的完备性的几个定理证明

实数的完备性的几个定理证明

实数完备性的六个基本定理，这里记录一下其中几个证明。

  • 基本公理：Dedekind 基本定理

    表述1：对 ℝ 中任何非空子集 𝔸 和 𝔹，若 ∀a ∈ 𝔸, ∀b ∈ 𝔹, a ≤ b，则 ∃c ∈ ℝ, a ≤ c ≤ b.

    表述2：如果 ℝ 两个子集满足如下性质：

    • 集合 𝔸 和 𝔹 都是非空的；

    • 𝔸 ∪ 𝔹 = ℝ；

    • 集合 𝔸 中的每一个数都小于 𝔹 中的任意一个数.

      这种分割 𝔸|𝔹 称为一个戴德金分割，对于任意这种分割，都存在这个分割的实数 α，这个数或者是下组 𝔸 的最大数，或者是上组 𝔹 的最小数.

  • 定理一：确界原理

    非空有上(下)界数集必有其上(下)确界.

  • 定理二：单调有界收敛定理

    单调数列收敛的充要条件是其为有界数列，进一步地，当数列 an 单调增时，$\lim\limits_{n \to \infty}a_n = \sup{\{a_n|n \in N^*\}}$，当数列 an 单调减时，$\lim\limits_{n \to \infty} = \inf{\{a_n|n \in N^*\}}$.

  • 定理三：Cauchy 收敛准则

    数列收敛的充要条件是它为柯西数列.

    柯西数列定义：对于数列 an，满足 ∀ϵ > 0, ∃N = N(ϵ) ∈ ℤ*, ∀m, n > N，有 |an − am| < ϵ.

  • 定理四：区间套定理

    设有闭区间列 {[an, bn]}，满足条件：

    • [a1, b1] ⊃ [a2, b2] ⊃ ⋯ ⊃ [an, bn] ⊃ ⋯;
    • $\lim\limits_{n \to \infty} (b_n-a_n) = 0$.

    则 ∃ 唯一的点 ξ，使得 $\xi \in \bigcap_{n=1}^\infty [a_n,b_n]$

  • 定理五：Bolzano 定理

    如果一个数列有界，那么他一定有一个收敛的子数列.

  • 定理六：Borel 有限覆盖定理

    如果开区间集合 𝕊 覆盖了闭区间 [a, b]，则 𝕊 中存在有限个开区间也覆盖了闭区间 [a, b]

还有其他很多定理，这里先给出前五个定理之间的证明。

来一些不等式、恒等式、定理的证明

发表于 2025-10-06 更新于 2025-10-15 分类于 线性代数 本文字数： 4.6k 阅读时长 ≈ 17 分钟

来一些不等式、恒等式、定理的证明

最近翻到了这些不等式、恒等式、定理这些东西，害怕忘掉这里简要记录一下。

首先先说明下这些定理。

  • Chebyshev 不等式

    • 对于序列 a1 ≤ a2 ≤ ⋯ ≤ an, b1 ≤ b2 ≤ ⋯ ≤ bn，则有：

    $$ \begin{flalign} \left(\sum_{i=1}^na_i\right)\left(\sum_{j=1}^nb_j\right)\leq n\sum_{k=1}^na_kb_k \end{flalign} $$

  • Cauchy 不等式

    • 设两组实数 x1, x2, ⋯, xn 和 y1, y2, ⋯, yn，则有：

    $$ \begin{flalign} \left(\sum_{i=1}^nx_iy_i\right)^2\leq\left(\sum_{i=1}^nx_i^2\right)\left(\sum_{i=1}^ny_i^2\right) \end{flalign} $$

  • Abel 恒等式

    • 记 $B_0=0,B_n=\sum_{i=1}^nb_i$，则： $$ \begin{flalign} S_n=\sum_{i=1}^na_ib_i=B_na_n-\sum_{i=1}^{n-1}B_i(a_{i+1}-a_i) \end{flalign} $$

  • 二项式定理

    • 对 ∀x ∈ ℝ, n ∈ ℕ*，有：

    $$ \begin{flalign} (1+x)^n=\sum_{i=0}^n{n\choose i}x^i \end{flalign} $$

  • 推广的 Bernoulli 不等式

    • 若 ai > −1, 1 ≤ i ≤ n, i ∈ ℕ* 且同号，则有：

    $$ \begin{flalign} \prod_{i=1}^n(1+a_i)\geq1+\sum_{i=1}^na_i \end{flalign} $$

  • Hölder 不等式

    • 设两组非负实数 x1, x2, ⋯, xn 和 y1, y2, ⋯, yn，并设 $p,q>1,\frac{1}{p}+\frac{1}{q}=1$，则有：

    $$ \begin{flalign} \sum_{i=1}^nx_iy_i\leq\left(\sum_{i=1}^nx_i^p\right)^\frac{1}{p}\left(\sum_{j=1}^ny_j^q\right)^\frac{1}{q} \end{flalign} $$

利用 C 语言实现已知三条边判断三角形形状并计算其面积

发表于 2025-10-05 分类于 计算机程序设计(C) 本文字数： 3k 阅读时长 ≈ 11 分钟

利用 C 语言实现已知三条边判断三角形形状并计算其面积

同样是布置的一道课后思考的算法题，我就顺便把这道题给写出来了。主要是代码量有点庞大，顺便还写了个冒泡排序算法。

这里简单介绍一下思路：

如何判断三角形形状，这里用到了余弦定理 $$ \begin{flalign} \cos C&=\frac{a^2+b^2-c^2}{2ab}\\ 当\cos C&>0,为锐角三角形\\ 当\cos C&=0,为直角三角形\\ 当\cos C&<0，为钝角三角形\\ \end{flalign} $$ 而分母 2ab > 0 恒成立，因此只需要判断分子 a2 + b2 − c2 与 0 的大小关系即可。

而这里就出现了一个问题，只需要一个角是直角或钝角，就是直角或钝角三角形，如果角 C 恰好是锐角就无法判断了吗？或者要判断三遍了吗？为了解决这个问题，这里我引入了排序算法，首先将序列 (a, b, c) 按照从小到大的顺序排列好，即消除其中的逆序对，就可以确定角 C 为三角形中最大的角，有一个定理：大角对大边，那么当边 c 最大时，角 C 也一定是最大的。

然后是计算三角形面积的问题，已知三边求面积，这里有两个思路，一种是 $$ \begin{flalign} \cos C&=\frac{a^2+b^2-c^2}{2ab}\\ \sin C&=\sqrt{1-\cos^2C}\\ S&=\frac{1}{2}ab\sin C\\ \end{flalign} $$ 其实是在计算 $\frac{1}{2}\left|\vec{a}\times\vec{b}\right|$，但这种方法被我淘汰了，因为可能出现大量的浮点数运算而且不够简洁。

这里我想到了 Heron 公式，又叫做秦九韶公式，专门用来计算已知三角形三边求面积的问题。 $$ \begin{flalign} S&=\sqrt{p(p-a)(p-b)(p-c)}\\ p&=\frac{a+b+c}{2}\\ \end{flalign} $$ 变得十分简洁了。

证明O'Stolz定理

O’Stolz 定理的简单证明

O’Stolz 定理的简单描述

该定理可以认为是 L’Hospital 法则在数列函数上的特殊情况。

$Th1: (\frac{*}{\infty})$型: 设数列 {an}, {bn} 满足以下条件:

  1. {bn} 严格单调递增,且 bn > 0,

  2. $\lim\limits_{n\to \infty}b_n = +\infty$.

则 $\lim\limits_{n\to \infty}\frac{a_n}{b_n}=\lim\limits_{n\to \infty}\frac{a_{n+1} - a_n}{b_{n+1} - b_n}=L$, 其中L为有限数或无穷大量.

$Th2:(\frac{0}{0}$)型: 设数列 {an}, {bn} 满足以下条件:

  1. {bn} 严格单调递减且 $\lim\limits_{n\to \infty}b_n=0$,

  2. $\lim\limits_{n\to \infty}a_n=0$.

则 $\lim\limits_{n\to \infty}\frac{a_n}{b_n}=\lim\limits_{n\to \infty}\frac{a_{n+1} - a_n}{b_{n+1}- b_n}=L$, 其中L为有限数或无穷大量.
